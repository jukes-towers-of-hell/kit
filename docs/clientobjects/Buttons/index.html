Buttons

ButtonConfigurations ​

ButtonPressOffset​

The offset of the button once it is pressed.

Default value: 0.75

Types:

  • number
  • Vector3
  • CFrame

PressedMaterial​

The material of the button once it is pressed.

Default value: Neon

Types:

  • EnumItem
  • string

Timer​

How many seconds the button will stay pressed. Only works if the value is above 0.

Default value: 0

Types:

  • number

DecimalPlaces​

The number of decimal places shown on the button's timer. Only works when Timer is active.

Default value: 0

Types:

  • number

TimerText​

The custom text for the button's timer. temporary special character will be replaced with the timer value.

Types:

  • string

HideGUI​

Whether the button timer will be visible.

Types:

  • boolean

ButtonActivatedPlatform ​

SetTransparency​

The transparency of the instance when it is enabled.

Default value: 0

Types:

  • number

FullHide​

Hides the instance when it is disabled.

Types:

  • tag

Invert​

Makes the instance enabled by default and disabled on button press.

Types:

  • tag

ColorOverride​

Overrides the button color that will activate the instance.

Types:

  • Color3

IgnoreCanCollide​

Prevents the instance's CanCollide property from being changed by buttons. Only works if the instance is a BasePart.

Types:

  • tag

IgnoreTransparency​

Prevents the instance's Transparency property from being changed by buttons.

Types:

  • tag

IgnoreAll​

A combination of IgnoreCanCollide and IgnoreTransparency.

Types:

  • tag
